# 🧠⚡ Math Blitz — מתמטיקה בלייז

A fast-paced cyberpunk math game for Israeli 6th graders. Practice converting between fractions, decimals, and percentages under time pressure!

## 🎮 Features

- **Adaptive difficulty** — gets harder as you improve, easier if you struggle
- **Accelerating timer** — 5% faster with every correct answer
- **Sound effects & vibration** — instant feedback on every answer
- **Local leaderboard** — compete against your own high scores
- **WhatsApp sharing** — challenge your friends with one tap
- **Full Hebrew RTL** — native right-to-left interface
- **Mobile-first** — designed for Android phones
